Codveda Data Analytics Internship Project
This repository serves as a professional portfolio documenting my journey through the Codveda Data Analytics internship, where I performed advanced data analysis, machine learning, and visualization tasks.

🚀 Levels Overview
Level 1: Data Cleaning & Exploratory Data Analysis (EDA)
Task: Cleaning raw datasets and performing initial analysis.

Detailed Process:

Loaded datasets and handled missing values using fillna(df.mean()).

Removed duplicates to ensure data integrity.

Performed initial EDA to understand data structure and feature distributions.

Key Output: Cleaned datasets and foundational analysis.

Level 2: Predictive Modeling & Clustering
Task 1: Linear Regression

Built a model to predict house prices.

Evaluated performance using R2 Score and Mean Squared Error to ensure model accuracy.

Task 2: K-Means Clustering

Implemented K-Means to segment house data into distinct groups based on features.

Used StandardScaler to normalize the data before clustering.

Key Output: Regression metrics and visual clustering maps (K-Means).

Level 3: Advanced Analysis (NLP & Classification)
Task 1: Sentiment Analysis (NLP)

Analyzed text data to determine the emotional polarity (Positive/Negative/Neutral).

Utilized TextBlob for natural language processing and visualized word frequency/sentiments.

Task 2: Classification (Churn Prediction)

Built a Random Forest Classifier to predict customer churn.

Handled non-numeric data using get_dummies and evaluated the model using a Confusion Matrix.

Key Output: Sentiment analysis reports and classification metrics (Accuracy & Confusion Matrix).

/Level_1: Contains data cleaning scripts and initial EDA visuals.
